Lecture 2 – HTML Basics

Defence Course - Day 2

1. What is HTML?

  • HTML stands for HyperText Markup Language.
  • It is the standard language to create web pages.
  • HyperText = linking between documents.
  • Markup = tags used to structure content.

2. World’s First Website

  • The first website was created in 1991 by Tim Berners-Lee.
  • It was purely text-based, without any styling.
  • It explained the concept of the World Wide Web and hyperlinks.

3. Hyperlinks

  • Hyperlinks connect one page to another.
  • They are created using the <a> (anchor) tag.
  • Example: <a href="page.html">Go to Page</a>

4. Markup Language

  • A markup language adds structure and meaning to text.
  • HTML is a markup language, not a programming language.
  • It uses tags like <h1>, <p>, <img> etc.

5. Setting Up a Code Editor

  • Common editors: VS Code, Sublime Text, Atom.
  • Install VS Code and create your first HTML file.
  • Save the file with .html extension.

6. Creating First Webpage

  • Every HTML file starts with <!DOCTYPE html>.
  • Main structure:
    • <html> → root element
    • <head> → metadata, title
    • <body> → visible content

7. Tags and Elements

  • Tags are keywords wrapped in <>.
  • An element = opening tag + content + closing tag.
  • Example: <p>Hello</p>

8. Headings

  • HTML provides 6 levels of headings: <h1> to <h6>.
  • <h1> = biggest, <h6> = smallest.
  • Use headings to structure your content properly.

9. Paragraphs

  • Paragraphs are written using <p> tag.
  • Used for normal text content.
  • Example: <p>This is a paragraph.</p>

10. Horizontal Rule and Line Break

  • <hr> = horizontal line, used to separate sections.
  • <br> = line break, moves content to the next line.

11. Lists

  • Ordered List (<ol>) → numbered list.
  • Unordered List (<ul>) → bullet points.
  • Each item inside a list is written with <li>.

12. Attributes

  • Attributes provide extra information to elements.
  • Written inside the opening tag.
  • Example: <a href="link.html">Click</a>

13. Adding Images

  • Use <img> tag to insert images.
  • Attributes:
    • src → image path
    • alt → alternative text
  • Example: <img src="photo.jpg" alt="My Photo">

14. Comments

  • Comments are ignored by the browser.
  • Used to add notes in code.
  • Syntax: <!-- This is a comment -->

15. Mini Project

  • Create a simple webpage with:
    • Headings
    • Paragraphs
    • Lists
    • Hyperlinks
    • Images

16. Wrap Up

  • HTML is the backbone of web development.
  • Learn the structure before moving to CSS and JS.
  • Practice by making small webpages.
