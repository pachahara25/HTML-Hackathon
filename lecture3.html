Lecture 3: Nested Lists & Tables

Defence Course - Day 3

1. Recap of Lists

  • Two main types of lists in HTML:
    • Ordered List (<ol>) → Numbered items (1, 2, 3...)
    • Unordered List (<ul>) → Bulleted items (•, ○, etc.)
  • Basic example: A simple country list (India, USA, England).

2. What is a Nested List?

  • A nested list is a list placed inside another list item.
  • Useful for showing hierarchy or categories (e.g., Country → Cities).
  • Correct placement: the nested <ul> or <ol> should always be inside its parent <li>.

3. Example: Countries with Nested Cities

Each country has its own sublist of cities.

  • India
    • Delhi
    • Mumbai
    • Goa
  • USA
    • California
    • Texas
  • England
    • London
    • Oval
    • Yorkshire

4. Introduction to Tables

  • Tables are used to arrange data in rows and columns.
  • Main tags used:
    • <table> → Creates the table.
    • <tr> → Table row.
    • <td> → Table data (cell).
    • <th> → Table header (bold, centered by default).

6. Key Learnings

  • Nested lists help create hierarchical structures.
  • Tables organize information neatly into rows & columns.
  • Always structure HTML properly to keep it valid and readable.
