Lecture 6: HTML Media

Defence Course - Day 6

1. First Principle: The Web is More Than Text and Images

The web should deliver not only text and images but also audio and video natively, without plugins. Earlier browsers required Flash, QuickTime, etc., which were inefficient and insecure.

2. The Core Problem

How to embed audio and video in a standardized, plugin-free way across all modern browsers?

3. The Logical Solution: <video> and <audio> Tags

HTML5 introduced dedicated tags for embedding media.

4. Part 1: The Basics - Getting a Video on the Page

<video src="my-awesome-video.mp4"></video>

Problem: Shows only the first frame, no play/pause/volume controls.

4.1. The controls Attribute

<video src="my-awesome-video.mp4" controls></video>

5. Part 2: Intermediate - Attributes

  • width & height → Set video dimensions
  • autoplay → Auto-plays (usually requires muted)
  • loop → Restarts video after ending
  • poster → Thumbnail image before play
<video src="video.mp4" width="640" height="360" controls></video>

6. Part 3: The <audio> Tag

<audio src="audio/theme-song.mp3" controls></audio>

7. Part 4: Advanced - Solving Real-World Problems

7.1. Problem #1: Different Browser Formats

<video controls poster="images/thumbnail.jpg">
  <source src="videos/my-video.webm" type="video/webm">
  <source src="videos/my-video.mp4" type="video/mp4">
  Sorry, your browser doesn't support embedded videos.
</video>

7.2. Problem #2: Accessibility - <track>

<video controls>
  <source src="video.webm" type="video/webm">
  <source src="video.mp4" type="video/mp4">

  <track src="captions_en.vtt" kind="captions" srclang="en" label="English">
  <track src="subtitles_es.vtt" kind="subtitles" srclang="es" label="Español">
</video>

8. Two Methods to Embed Video

8.1. Example 1: Professional Way (<source>)

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

8.2. Example 2: Simple Way (src Attribute)

<video src="a.mp4" controls></video>
